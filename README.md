## 🧠 Profile

- 🔭 **Current Focus:** Architecting **Hi-MambaSR**, a hierarchical state-space refinement model for latent diffusion super-resolution.
- 🌱 **Optimization:** Specializing in optimizing deep learning models for consumer-grade hardware (6GB GPUs) utilizing gradient checkpointing and 8-bit precision.
- 💼 **Experience:** Associate Software Engineer at **Blackbaud** | Former Data Science & Analytics Intern at **Corteva Agriscience** (Developed *AgriBot*).
- 🎓 **Education:** B.Tech in Computer Science & Engineering (8.81 CGPA) at **KIIT**.
- 💬 **Tech Interests:** Computer Vision, GANs, Diffusion Models, PyTorch Lightning, and State-Space Models.
